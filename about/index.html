我想说

个人始终坚信,所有不以实际入侵和防御为目的的安全研究都是耍流氓 [ 目标明确 ]

没有成功付诸实践的 “假如”, 也仅仅只是自己一厢情愿的意淫 [ 重在实践 ]

与其浪费大量时间随波逐流,不如静下心来,从 0 到 1 ,人云亦云基本不会有出路,我想要的只是相对本质,相信你也一样 [ 返璞归真并不等于闭门造车 ]

所有的复杂都是简单的堆积 “根基不牢,地动山摇”,好高骛远,最后害的,始终是自己 [ 不积跬步,无以至千里 ]

nothing,doing …… [ 少说多做 ]

不用太在意别人说什么,学会为自己而活,做好你自己,并时刻清楚你自己想要的到底是什么即可 [ 足够专注 ]

把会学习变成一种习惯,并好好把它坚持下去,你将受益终生 [ 不停刷新自己的知识层面 ]

善于站在前辈们的肩膀上,在现有的优秀经验基础上再根据自己的实际需求不断改进,相信你会走的更稳 [ 善于总结 ]

其实,并不一定非要自己有多么高的智商,始终坚信,足够的勤奋好学,会弥补这一切的先天不足,只要你愿意,这些都不是问题 [ 足够自信 ]

最后,再跟大家分享一句很久之前在一位前辈博客上看到的话 “我所说的我都做过“ ,至今对我触动依然非常大

放弃那些并不能给你带来什么的 虚荣 和所谓的 名号,id… ,相信你会活的更洒脱,也会得到更多实实在的东西

在你自己还没真正达到能立马指哪打哪的能力之前,请先学会夹着尾巴做人,想办法把自己当下手头里的事情做好即可

相信真正的价值都是来源于实实在在的贡献

以上这些这段话来自于一个前辈的！对我影响非常大，所以记录到博客！
